# Open WebUI for Raycast

Chat with your Open WebUI instance directly from Raycast.

## Setup

1. **Server URL**: Your Open WebUI instance URL (e.g., `http://localhost:3000`)
2. **API Key**: Get your API key from Open WebUI → Settings → Account

## Features

- **Chat**: Start conversations with any model available in your Open WebUI instance
- **New Chat**: Clear conversation and start fresh
- **Upload File**: Upload files for RAG (Retrieval Augmented Generation)
- **Browse Files**: View your uploaded files

## Requirements

- An Open WebUI instance (self-hosted or cloud)
- A valid API key from your Open WebUI account
